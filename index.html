Consejos:

Copia y ajusta componentes de getbootstrap.com.

Si estás viendo un ejemplo de https://getbootstrap.com/docs/5.1/examples/, puedes usar el inspector (clic derecho para abrir este panel) y luego seleccionar la sección que quieres en la pestaña "Elements". Luego ve a copiar, y copia el elemento para agregarlo rápidamente a tu proyecto.

Para subir cambios al sitio web, recuerda esta secuencia:

  • git add .
  • git commit -m "lo que modificaste"
  • git push

Para ver cambios de tus compañeros de equipo:

  • git pull

Para personalizar elementos, usa el archivo style.css ubicado en assets/css.
